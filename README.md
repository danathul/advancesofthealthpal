# HealthPal - Digital Healthcare Platform

## Overview
Brief description of the project

## Tech Stack
- Spring Boot 3.3.2
- MySQL 8
- Java 22
- Swagger UI

## Architecture
Layered architecture (Controller → Service → Repository)

## Setup Instructions
1. Clone repo
2. Create database: healthpal_db
3. Update application.properties
4. Run: mvn spring-boot:run
5. Access Swagger: http://localhost:8081/swagger-ui.html
